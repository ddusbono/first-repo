# First-Repo

github과 git을 배우기 위해 태어난 나의 첫 저장소.

See [Demo](https://github.com)

## Prerequisites

- Pytohn^3.12
- pandas==1.0.0

## How to Start

'''shell
 $ git clone {repo addr}
 $ cd {repo addr}
 $ pip install -r requirements.txt
 $ python main.py
'''
